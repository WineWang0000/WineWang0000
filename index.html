五种排序算法

Posted on 2019-03-03

什么是排序算法，需要有一下几点：

1.输入：计算机必须有0个或1个以上的输入量
2.输出：1个或以上的输出量
3.明确性：算法的描述必须无歧义
4.有限性：在有限的时间内破解
5.有效性： 又叫可能性，能够被计算机完成

算法的分类

  • 分治法
  • 动态规划法
  • 贪婪算法
  • 线性规范法
  • 简并法

    分治法是前端常用的

    下面讲一下有哪五种排序算法

    1.冒泡排序：

    假如有一组数组，让你按大小排序，人脑当然很简单就看出来，那计算机怎么排的呢？
    俩俩比较，比较第第1和第2个数，谁小谁拎前面排着，再比较第2个和第3个，2小就不用动，3小就放2前面，继续比较3和4，谁小谁放前面，这样一轮一轮的算。得出最后的结果。

    2.选择排序：

    找最小，放最前面，再找第二小，放第一小后面，总之就是找最小的往前放。

    3.插入排序：

    这个有点像起牌，最小放前面，下一个比它大的插到最小后面，以此类推。

    4.基数排序：


    如图，把每个不同的数都放在不同的块(我称为块，好理解)，在块里放好后再拿出来进行排序。

    5.随机快排：

    12 3 7 89 66这样一组数字，随机选个数比如7，7和12比12大，所以7在12前面，和3比7大，3放7前面，再和89比，比7大，位置对，不用换，再和66比，位置对，不用换。这样下来是 3 7 12 89 66。再随机选，比如12，12和3比3小位置正确和7比7小位置正确。就这样一直比。直到排完。(运气好能很快排完，常用这个)

CSS介绍

什么是CSS？

定义：CSS 指层叠样式表 (Cascading Style Sheets)，是一种用来表现 HTML 文档样式的语言，样式定义如何显示 HTML 元素，是能够真正做到网页表现与结构分离的一种样式设计语言。样式通常存储在样式表中，外部样式表通常存储在 CSS 文件中，多个样式定义可层叠为一。
通俗说就是样式，就像人身上的装饰。用<style></style>包裹。目前使用的是CSS2.1版本

CSS语法

CSS两个主要的部分构成：选择器，以及一条或多条声明。如下：

h1是选择器一种，color和background叫属性，后面的是属性值。CSS要写在head标签里，HTML写在body标签里。

CSS如何引入

  • 内联样式

直接写在HTML标签里，但是这种写法现在不用。再看第二种。

  • style标签

这个就是现在的写法，注意标点符号，对齐问题。

  • CSS link 外部文件
    假如你做的项目很大，需要写很多CSS，你也可以创建一个a.css文件(可以为任何名字，后缀必须是css)，把css内容写在文件内。然后在head里添加这样一段 <link rel=”stylesheet” href=”·/a.css”>
    意思是引入当前a.css文件,第一个点代表当前文件夹的。

a.css文件里面直接这样写就行

  • @inport url = “·/a.css”
    创建一个b.css文件，里面同样是css样式。再在a.css文件内第一行写入@inport url = “·/a.css” 意思是在引入a.css之前引入下b.css
