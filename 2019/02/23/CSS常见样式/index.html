9.display属性 指定元素的显示类型

了解这个先要讲解下块级元素和行内元素。

行内元素：

自己的独立空间，它是依附于其他块级元素存在的，因此，对行内元素设置高度、宽度、内外边距等属性，都是无效的。比如a标签。样式是行内元素，因此对a元素设置高度、宽度等属性，都是无效的。

块级元素

比如div，p，li，img等默认情况下的display属性值就是“block”。所以对于链接a来说只能这样：display: inline;改为display: block;强制将它改成块元素。看代码。
