赋值运算符

X = Y,即把Y赋值给X，记住是后面的赋值个前面的

其他的还有：
X += Y等同于X = X + Y
X -= Y等同于X = X - Y
X = Y等同于X = X Y
X /= Y等同于X = X / Y
X加上Y之后再赋值给X

布尔运算符（true or false）

符号表示 名称 返回值
！ 取反 true/false
&& 且运算符 true/false
ll 或运算符 true/false

1.!取反

2.且运算符

先看第一项为true，再看第二项不为true，返回第二个结果false。如果两边都为true，则结果为true
3.或运算符

有一个为true则为true,0转换成布尔是false，再看第二个1为true，返回第二个true的结果。第二个例子，3位true所以不用看第二个，返回true。

三元运算符

条件判断 如果第一个表达式的布尔值为true，则返回第二个表达式的值，否则返回第三个表达式的值。

t的值为true，所以返回hello。0是false，返回world。
