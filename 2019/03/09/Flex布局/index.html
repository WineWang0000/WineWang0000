Flex布局

Posted on 2019-03-09

flex之前用什么

  • position relative或者absolute
  • float和clearfix
  • 负margin
  • normal flow(正常流，又名文档流)
  • display inline-block

    flex的作用：

  • 块级布局注重垂直方向，行内侧重水平方向。但flex与方向无关。
  • 可实现空间自动分配，自动对齐。
  • 适用于简单的线性布局，复杂的使用gird布局。

    在父元素上加display：flex；

    flex container(父元素)的属性

    flex-direction属性指定了内部元素是如何在 flex 容器中布局的，定义了主轴的方向(正方向或反方向)。

justify-content主线轴对齐方式

align-items侧轴对齐方式

父元素上一定加上display:flex;其他一个个试就能明白意思。就不多说了。
