类型转换

类型转换一般是用number，string，boolean函数手动将各种类型的值转换为数字，字符串，布尔。

1.Number

先看转换成数字

类型 转换成数字
undefined NaN
null 0
string “abc”→NaN,”123”→123
boolean true→1，false→0

2.转换成Boolean

类型 转换成布尔
null false
undefined false
string 空字符串（””）为false，其他都为true
NaN false
number 0→false，其他为true
object true
boolean boolean
  • 五个永远为false的，0，null，undefined，NaN，空字符串。
  • object 都是true
  • 空字符串和空白字符串是不一样的哦！’’里面没有内容，连空格都没有，’ ‘空白字符串中间有空格，不是空的。为true。

    3.string

    |字符串|还是原来的值|
    |布尔|字符串布尔,”true”,”false”|
    |null|”null”|
