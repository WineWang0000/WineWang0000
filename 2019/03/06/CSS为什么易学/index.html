CSS为什么易学

Posted on 2019-03-06

之前我们还说CSS变态，现在又说它易学是为啥么嘞？请听本回分解。

虽然CSS够变态，但是我们学会一些小套路，也能应对平时的需求。

1.水平居中

块级元素：

在宽确定的情况下，margin-left: auto;,margin-right: auto;即可水平居中。宽不确定的情况下，margin-left：xpx;,margin-right: xpx;.x设置为你需要的，这样也能实现水平居中。

内联元素：

这个很简单，只需在父元素上text-align: center;即可。

2.垂直居中

只需要给padding-top: 20px;padding-bottom: 20px;设置同样的距离，自然就居中了。

3.绝对居中

三件套：display: flex; justify-content: center; align-items: center;

4.布局

一般记住float和flex布局就行，在重复一遍，不要忘记去除浮动。
